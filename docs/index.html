プリズナートレーニング

プリズナートレーニングは、元囚人のポール・ウェイドが監獄で培った自重トレーニング（キャリステニクス）の技術を伝える本、 およびそのトレーニング法です。 原題は『コンビクト・コンディショニング』ですが、日本語版の書籍では『プリズナートレーニング』となっています。

ビッグ６

プリズナートレーニングでは、プッシュアップ（腕立て伏せ）、スクワット、プルアップ（懸垂）、レッグレイズ、ブリッジ、ハンドスタンド・プッシュアップ（逆立ち腕立て伏せ）の６つの種目を扱います。 各種目内に10段階のステップがあり、各ステップで目標とするセット数、レップス数が示されています。

例えば、プッシュアップのステップ１は「ウォール・プッシュアップ」（壁に向かった腕立て伏せ）で「初心者の標準」が10レップスを1セット。 ステップ５が「フル・プッシュアップ」（いわゆる普通の腕立て伏せ）で、「上級者の標準」が20レップスを2セット、という具合です。

ひとつひとつステップを上げていくことで、最終的には超人的なステップに到達できるとされています。 プッシュアップのステップ10は「ワンアーム・プッシュアップ」（片手腕立て伏せ）です。

PT Webでは、これら6種目×10ステップについて、セット数とレップ数を記録できます。

アプリの機能

レーダーチャートで達成度を確認

ビッグ6の達成度をレーダーチャートで表示します。1つのステップの記録を5日分以上入力すると、チャートにプロットされます。

プリズナートレーニングでは筋力に自信のある人もステップ1から順にトレーニングを重ねていくことが推奨されています。 レーダーチャートはステップごとにプロットされるため、ステップを進めると年輪のような表示になります。

トレーニングの頻度を可視化

直近数日間のトレーニング実施日が方眼のマスに着色されることで可視化されます。 色はステップによって異なり、レップス数に応じた3段階の濃淡があります。表示される日数は端末の横幅によって変わります。

Weekly Scoreで総負荷量をコントロール

直近7日間のトレーニング記録に応じたスコアを表示します。スコアはレップス数×ステップ数で計算されます。1週間のトレーニングの総負荷量の目安となります。

ウォール・プッシュアップ 50レップスを3セットなら50 x 3 x 1 = 150、フル・プッシュアップ 20レップスを2セットなら20 x 2 x 5 = 200です。

棒グラフでトレーニングの成果がわかる

種目ごとのレップス数を棒グラフで表示します。グラフは各ステップの「上級者の標準」のレップス数を100とした相対的な値です。 ステップによってグラフの色が異なります。

Twitter連携

トレーニング内容をTwitterに投稿できます。 当日のトレーニング内容を「保存」すると、「Tweetできます」というダイアログが表示されます。Tweetアイコンを押すと、 種目名、ステップ、各セットのレップス数とハッシュタグ「#プリズナートレーニング」が入力された状態で投稿フォームが開くので、必要に応じて追記・編集を加えて、投稿してください。

過去の記録を編集したときはTweetのダイアログは表示されません。

データのバックアップ

トレーニングのデータは端末内に保存されます。 Twitterアカウントを使って認証を行うと、PT Web上のデータベースと端末の間でトレーニングデータを同期できます。 PT Web上にデータをバックアップしておけば、別の端末にデータを引き継ぐことができます。なお、同期できるのは直近の180日分のデータです。

アカウント認証ではTwitterのAPIを利用しています。パスワードがPT Webに送信されることはありません。 PT Webに保存されるのはユーザーの識別子とトレーニングのデータだけです。

PCやスマホにインストール

PT WebはPWAです。PWAはPCやスマホにインストールしてネイティブアプリのように利用できます。

iOSの場合

  1. SafariでPT Webを開き、ツールバーから「共有」シートを表示します。
  2. 「ホーム画面に追加」でインストールできます。

Androidの場合

  1. Chromeの右上メニューから「ホーム画面に追加」でインストールできます
